# LLM Reasoning Analysis

This project explores the reasoning capabilities of Large Language Models (LLMs) on STEM-related analytical problems.

The goal is to evaluate how well modern AI systems perform on tasks involving:

- mathematical reasoning
- logical inference
- conceptual explanation
- multi-step problem solving

## Project Components

prompts.md  
Contains a dataset of reasoning prompts designed to test analytical capabilities.

evaluation_rubric.md  
Defines a scoring rubric used to evaluate AI-generated responses based on accuracy, reasoning quality, and clarity.

## Evaluation Criteria

Responses are evaluated across three dimensions:

1. Accuracy
2. Reasoning Quality
3. Clarity of Explanation

Each dimension is scored from 1–3, producing a total score out of 9.

## Purpose

Large language models often produce fluent explanations but may struggle with deeper reasoning tasks.  
This project documents a small evaluation framework for analyzing those strengths and limitations.
